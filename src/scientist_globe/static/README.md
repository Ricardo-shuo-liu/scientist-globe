# 全球计算机科学家 3D 互动地球仪 (Vibe Edition)

这是一个炫酷、丝滑、开箱即用的 3D 互动前端项目。通过 3D 地球仪全方位展示全球顶尖计算机科学家的分布与其核心贡献。

## 🚀 运行指南 (重要！)
1.  **切勿直接双击打开 index.html**：由于浏览器对 ESM 模块的安全限制，直接通过 `file://` 协议打开会导致脚本无法加载。
2.  **使用 Python 启动 (推荐)**：
    -   在你的电脑上，打开终端 (CMD 或 PowerShell)。
    -   依次运行以下**两行**命令：
        ```bash
        cd scientist-globe
        python -m http.server
        ```
    -   **关键点**：运行后**不要关闭终端窗口**，也不要按 `Ctrl+C`。
    -   手动打开浏览器访问：`http://localhost:8000`
3.  **其他方式**：
    -   如果你有 **VS Code**，安装 **Live Server** 插件，右键 `index.html` 选择 "Open with Live Server"。

## ✨ 交互说明
-   **拖拽**：鼠标左键旋转地球，右键平移。
-   **缩放**：滚动鼠标滚轮缩放地球。
-   **点击标记点**：点击地球表面的发光点，查看科学家的详细生平。
-   **自动驾驶**：如果 5 秒无操作，地球会自动开始缓慢自转。
-   **关闭面板**：点击右上角关闭按钮、点击地球空白处或按 `ESC` 键。

## 🛠️ 自定义扩展
-   **修改数据**：直接编辑 `js/data.js`，按照数组格式添加或修改科学家信息即可，无需触碰核心渲染逻辑。
-   **调整样式**：在 `css/style.css` 中修改变量，即可一键更换主题色（如改为赛博朋克粉、黑客绿等）。

## 📦 技术栈
-   [Three.js](https://threejs.org/) - 强大的 3D 渲染引擎
-   [Font Awesome](https://fontawesome.com/) - 图标库
-   纯原生 JS (ES6+) - 零构建工具，零依赖
